## Tira Insights Design Theme
Apply the following design system exactly — sourced from the official Tira Insights editorial stylesheet.

### Palette
| Token      | Hex       | Use                                              |
|------------|-----------|--------------------------------------------------|
| paper      | #F6F2E9   | Figure background — warm cream                   |
| paper-2    | #EFE9DA   | Axes / chart frame background                    |
| paper-3    | #E7E0CC   | Default / neutral bars, ghost elements           |
| ink        | #1A1A1A   | All titles, axis labels, primary text            |
| ink-soft   | #2A2A2A   | Body / annotation text                           |
| muted      | #5A5A5A   | Captions, subtitles, tick labels                 |
| subtle     | #8C8C8C   | Footer, metadata                                 |
| rule       | rgba(26,26,26,0.14) | Axis lines, borders                   |
| rule-soft  | rgba(26,26,26,0.08) | Gridlines                             |
| gold       | #A87C30   | Positive sentiment · things to grow · highlights |
| gold-deep  | #826021   | Secondary positive emphasis                      |
| gold-soft  | rgba(168,124,48,0.10) | Positive fill / card background      |
| blush      | #8A4F48   | Negative sentiment · risk · pain points          |
| blush-soft | rgba(138,79,72,0.10)  | Negative fill / card background      |
| sage       | #4F7259   | Healthy / neutral-positive · Android platform    |
| sage-soft  | rgba(79,114,89,0.10)  | Healthy fill / card background       |

### Semantic colour rules (follow strictly)
- **Android platform** → sage `#4F7259` (healthier platform)
- **iOS platform** → blush `#8A4F48` (problem platform)
- **Positive sentiment** → gold `#A87C30`
- **Negative sentiment** → blush `#8A4F48`
- **Neutral sentiment** → paper-3 `#E7E0CC`
- **Rating >= 4** → sage; **Rating >= 3** → gold; **Rating < 3** → blush
- Gridlines: color `#1A1A1A`, alpha 0.08, linewidth 0.6
- All spines: color `rgba(26,26,26,0.4)`, linewidth 0.8; top/right spines hidden
- Tick label color: muted `#5A5A5A`, fontsize 8

### Typography
- Figure title / section titles: **Georgia** bold, ink `#1A1A1A`
- Axis labels, annotations, KPI labels: **Inter** (or DejaVu Sans as fallback), ink-soft
- Eyebrow labels (small caps above charts): Inter 700, uppercase, letter-spacing wide, gold `#A87C30`
- Caption / footnote text: Inter, subtle `#8C8C8C`, italic
- KPI numbers: Georgia bold, large, colored by semantic role
